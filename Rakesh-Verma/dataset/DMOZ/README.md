# Dados do DMOZ
Este dataset foi obtido seguindo as instruções do artigo [Web classification using DMOZ](https://github.com/drypb/pescaria-et-al/blob/Rakesh-Verma/Rakesh-Verma/dataset/DMOZ/Lisa_Hoek--Web_classification_using_DMOZ.pdf).
Ele contém URLs legitimas coletadas entre 1998 e 2017. O arquivo .rdf.u8 disponível no Wayback Machine já foi parseado e comprimido para conter apenas uma lista de URLs ([dmoz.tar.bz2](https://github.com/drypb/pescaria-et-al/blob/Rakesh-Verma/Rakesh-Verma/dataset/DMOZ/dmoz.tar.bz2)).
